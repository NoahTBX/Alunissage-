# Alunissage-
Le but du projet est de simuler un alunissage o(*°▽°*)o
Description:
Ce projet est une simulation interactive d'atterrissage sur la Lune. Le joueur controle la quantite de carburant a bruler a chaque etape pour ralentir la descente du module lunaire, en tenant compte de la gravite lunaire et de la physique de base de la propulsion rocket.
Le programme est ecrit en Python et utilise Tkinter pour une interface graphique simple. Il inclut deux modes :

Mode Immersif : Avec des delais de 10 secondes entre les actions pour simuler le temps reel.
Mode Rapide : Sans delais, avec des parametres initiaux aleatoires.

Fonctionnalites:

Calculs physiques approximatifs : Gravite lunaire (1.62 m/s²), propulsion basee sur la conservation de la quantite de mouvement.
Interface GUI : Affichage en temps reel de la hauteur, vitesse et carburant. Representation graphique basique (canvas Tkinter).
Validation des entrees et gestion des erreurs.
Resultat final : Succes si vitesse d'impact ≤ 10 m/s.
